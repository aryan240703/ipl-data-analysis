 📊 IPL Data Analysis (2008–2022)

This project explores **Indian Premier League (IPL) match data from 2008 to 2022** using Python, Pandas, Matplotlib, and Seaborn. The dataset includes match details such as teams, venues, toss decisions, winners, margins, and player awards. The analysis highlights team performances, toss impacts, venue trends, and standout players.

 🔑 Key Features
- **Data Cleaning & Preprocessing**  
  - Converted date columns to `datetime` format  
  - Extracted year, month, and day for time-based analysis  
  - Handled missing values and standardized columns  

- **Exploratory Data Analysis (EDA)**  
  - 📈 **Most Successful Teams**: Count of wins across seasons  
  - 🎲 **Toss Decision Impact**: Effect of batting vs. fielding first on match outcomes  
  - 🏟️ **Venue Analysis**: Top venues and their most frequent winning teams  
  - 🏅 **Player of the Match**: Top 10 players with the most awards  
  - ⚔️ **Batting First vs. Chasing**: Comparison of wins by runs vs. wickets  

- **Visualizations**  
  - Bar plots, count plots, and comparative charts for clear insights  
  - Highlighted trends across seasons and venues  

 📂 Tech Stack
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)  
- Google Colab for interactive analysis  

 📊 Insights
- **Mumbai Indians** and **Chennai Super Kings** emerge as the most successful teams.  
- Toss decisions show varying impacts, with chasing often proving advantageous.  
- Certain venues like **Wankhede Stadium** and **Eden Gardens** consistently host high-impact matches.  
- Players like **AB de Villiers, Chris Gayle, and Rohit Sharma** dominate Player of the Match awards.
